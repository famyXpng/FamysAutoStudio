## 🚀 Overview

**Famy's Auto Studio** is an advanced desktop automation platform built specifically to be entirely window-aware and non-intrusive. Unlike standard macro recorders that take over your mouse and keyboard, Famy's Auto Studio interfaces directly with the Windows API (`user32.dll`) to send simulated inputs *directly* to background windows. 

This means you can auto-farm, bot, or script applications (like Roblox, Minecraft, or Notepad) while they are completely hidden or minimized to the System Tray, allowing you to use your computer normally!

## ✨ Features

*   **⚡ Auto Tap (`.ftap`)**: High-speed, customizable interval tapping with Humanized Jitter capabilities. 
*   **⏺ FamyScript Macros (`.fmac`)**: A custom-built, lightweight scripting language (`type()`, `press()`, `wait()`, `notify()`) that executes synchronously against target windows.
*   **⏱ Task Scheduler (`.ftsk`)**: Fully autonomous background watchers. Trigger actions based on time, file creation, or when specific windows are opened/closed.
*   **⬇️ Force System Tray**: Instantly minimize and hide any running application into your System Tray to reduce GPU/CPU overhead while the bot continues to run.
*   **🎨 Premium Glassmorphic UI**: Beautifully designed Light/Dark aesthetic interface.

## 📦 Installation (Alpha)

This software is currently in Closed Alpha. Source code is not publicly available.

1. Navigate to the **[Releases Tab](../../releases)** on this repository.
2. Download the latest `FamysAutoStudio_Installer.exe`.
3. Run the installer wizard to set up the application on your Windows machine.
4. Double-click any `.ftap`, `.ftsk`, or `.fmac` file to instantly launch and import your presets!
